# bulkchain
node based utilities for systems integration

### installation
start your bitcoind:  
``` bitcoind --daemon [--txindex]```  


install bulkchain:
```git clone https://github.com/m0dul0/bulkchain.git  
cd bulkchain  
npm install  
npm test```  

### usage:
bulkchain-cli is currently pie-in-sky.  
appeal to bulkchain-test.js for examples.
